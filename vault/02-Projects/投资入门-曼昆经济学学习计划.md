# 投资入门-曼昆经济学学习计划

## 项目目标
- 用 4 周建立“经济学思维 -> 投资决策语言 -> 个人判断框架”。
- 形成可复用输出：问题清单、观点记录、二次精读笔记、术语卡片。

## 阶段目标（4 周）
1. 第 1 周：经济学基本思维
- 目标：掌握机会成本、边际分析、激励、比较优势。
- 产出：10 条碎片笔记 + 1 页“我如何做决策”的总结。

2. 第 2 周：市场机制与价格信号
- 目标：掌握供需、弹性、税收与管制对市场的影响。
- 产出：2 个现实案例（如房租、油价）分析。

3. 第 3 周：宏观指标与投资环境
- 目标：理解 GDP、通胀、失业、利率与资产价格关系。
- 产出：1 页“宏观变量如何影响股债现金配置”草图。

4. 第 4 周：从经济学到投资框架
- 目标：把前 3 周概念映射到你的投资流程。
- 产出：个人《投资前检查清单 v1》。

## 学习循环（每轮）
1. 带问题阅读（先读，不急总结）。
2. 你写碎片与观点（含“我不确定的地方”）。
3. 我做纠偏和追问。
4. 二次精读（只回看关键段落）。
5. 固化到知识库（知识页/术语卡/复盘）。
