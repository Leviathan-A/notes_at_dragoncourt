# 快速流程：Inbox 转知识

## 目标
把 `00-Inbox` 的临时记录，转化为可复用的项目资产或知识条目。

## 5 步流程
1. 浏览 `00-Inbox`，把条目标注为：项目 / 知识 / 资源 / 丢弃。
2. 项目类内容移入 `02-Projects` 对应页面。
3. 可复用结论抽取到 `03-Knowledge`，补“适用场景”和“边界”。
4. 在日报中补上迁移记录与双链。
5. 清理 Inbox 已处理项，保持收件箱轻量。

## 判定原则
- 可复用：进 `03-Knowledge`
- 有截止日期：进 `02-Projects`
- 长期职责：进 `04-Areas`
- 仅参考：进 `05-Resources`
